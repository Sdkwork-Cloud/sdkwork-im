- `S09 = CCP 与 runtime/business model 对齐`

## 已兑现能力
- `protocol-governance` 已发布 `businessPolicyVocabulary`。
- conversation 聚合已落地 `ConversationPolicy / policy_epoch`。
- runtime 已落地：
  - `ApplyConversationPolicyCommand`
  - `conversation.policy_applied`
  - replay 恢复
  - 建会后即时绑定策略
  - history query auth-context entrypoint
  - `GET /im/v3/api/chat/conversations/{conversation_id}/messages`
- `capability_flags` 已真实约束：
  - `message.reaction`
  - `message.pin`
- `history_visibility` 已真实约束最小历史读路径：
  - `joined`：非成员 `403`
  - `world_readable`：同租户已鉴权非成员 `200`
- governance 当前只发布：
  - `historyVisibilityModes = joined / world_readable`
- 未实现模式已被显式阻断：
  - domain normalize 拒绝 `invited/shared`
  - HTTP 建会拒绝 `invited/shared`
  - runtime 读路径若遇 legacy `invited/shared` 返回 explicit unsupported

## 未兑现能力

- `shared` 的 shared-channel / external durable truth。
- `retention_policy_ref` 与 archive/restore/retention owner 的联动。
