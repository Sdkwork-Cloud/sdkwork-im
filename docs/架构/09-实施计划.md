# Craw Chat Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 构建一套支持 SaaS 与私有化双交付的 Rust 即时通信服务端最小可运行版本，并在文档冻结的前提下逐步扩展到流式通信、RTC 信令、文件资源、通知与自动化能力。

**Architecture:** 采用控制面 / 数据面 / 运维面分层，数据面按会话路由和 commit journal 驱动，查询视图按 projection 构建，基础设施通过 Ports and Adapters 接入。

**Tech Stack:** Rust, Tokio, Axum/Hyper, Tonic, OpenTelemetry, default adapters for Redpanda / CockroachDB / ScyllaDB / Valkey / etcd / S3.
