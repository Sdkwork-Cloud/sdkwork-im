# 152CJ Loop36补充 - 2026-04-10

- `S05` runtime-dir social durable truth 当前再向前收敛一格：
  - replay key 已按 `(tenant_id, event_id)` 识别 committed social event
  - same-event retry 若命中已提交 commit，则直接返回 committed result
  - same-event replay 不再重复追加 journal，并会 best-effort 修复 snapshot cache
- 当前客户端语义从“truth 已提交但确认不清”收敛为“首次响应可能 unavailable，但 same-event retry 可确认 truth”；这仍不是完整 `atomic tx + repair` 闭环。
- 因仍缺 `atomic tx boundary + failpoint/repair evidence + operator-grade replay/repair`，`S05` 仍只能标记为 `not_closed / local_closure`。
