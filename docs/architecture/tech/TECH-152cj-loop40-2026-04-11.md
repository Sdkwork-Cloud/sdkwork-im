# 152CJ Loop40补充 - 2026-04-11

- `S05` runtime-dir social durable truth 再向前收敛一格：
  - `control-plane-api repair-social-runtime-dir --runtime-dir <path> [--json]` 提供了跨进程 standalone 的 social journal replay/repair CLI
  - operator 可在不启动 control-plane 服务的前提下，直接从 `social-commit-journal.json` 重建 `social-state.json`
  - repair 结果会显式输出 aggregate counts，便于审计与人工确认
- 当前 `S05` 的剩余主缺口已不再是 standalone repair 能力本身，而是：
  - `atomic multi-file tx`
  - 既有 `repair-runtime-local.*` 与 social standalone CLI 的统一 operator surface
- 因 social 写路径仍非真正 multi-file 原子事务，且 operator surface 尚未完全统一，`S05` 仍只能标记为 `not_closed / local_closure`。
