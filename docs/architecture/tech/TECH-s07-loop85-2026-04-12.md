# S07 Loop85 Addendum - 2026-04-12

## 1. Main Conclusion
- Loop85 extends same-owner stale-lease normalization into the `trigger_unconfigured` targeted republish path.
- A same-owner stale republish attempt no longer returns `trigger_unconfigured` while leaving the request serialized with expired lease metadata.

- `services/control-plane-api/tests/social_external_collaboration_test.rs`
