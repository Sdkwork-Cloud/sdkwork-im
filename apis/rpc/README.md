# RPC API Contracts

## Purpose

Author-owned gRPC/protobuf contracts for Sdkwork IM communication services.

## Owner

SDKWork Chat maintainers.

## Allowed Content

- `.proto` schema files under `sdkwork/`.
- RPC examples, changelogs, and validation fixtures promoted from SDK workspaces.

## Forbidden Content

- Generated RPC SDK language output.
- Runnable RPC servers or service implementations.

## Related Specs

- `../../sdkwork-specs/RPC_SPEC.md`
- `../../sdkwork-specs/RPC_SDK_WORKSPACE_SPEC.md`
- `../../sdkwork-specs/API_SPEC.md`

## Verification

Run `pnpm run test:rpc-contract` from the repository root.
