# Examples

## Purpose

Runnable examples, integration examples, sample configs, and SDK/API usage examples for Craw Chat.

## Owner

SDKWork Chat maintainers.

## Allowed Content

- Consumer-facing SDK examples.
- API usage examples and sample non-secret config.
- Runnable integration examples that are not production app roots.

## Forbidden Content

- Production runtime state.
- Secrets, tokens, customer data, generated SDK transport output, or caches.
